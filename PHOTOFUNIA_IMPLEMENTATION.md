## How It Works

### API Integration
The commands use the **PhotoFunia API** from:
```
https://api.giftedtech.co.ke/photofunia/
```

Each effect endpoint accepts a `text` parameter:
```
https://api.giftedtech.co.ke/photofunia/{effect}?text={text}
```

### Command Flow
1. User types: `.neon Hello World`
2. Bot parses the command and extracts text
3. Bot makes API call to PhotoFunia endpoint
4. API returns image URL
5. Bot sends image with caption showing effect name

### Response Format
```
[Generated Image]

*HELLO WORLD*

_Effect: Neon Writing_

GENERATED BY GODSZEAL XMD
```
