## Deploy on Katabump with Session ID

### Step 4: Deploy on Katabump with Session ID
**To deploy on Katabump, follow these steps:**

1. **Get your Session ID:**  
   Visit [https://gzpair.vercel.app](https://gzpair.vercel.app) and follow the on-screen instructions to obtain your session ID.
   
2. **Deploying on Katabump:**  
   - Open your terminal.  
   - Use the following command to deploy:
     ```bash
     deploy-command --session-id <YOUR_SESSION_ID>
     ```
   - Replace `<YOUR_SESSION_ID>` with the session ID you obtained earlier.
   
3. **Confirm your deployment:**  
   - Once the command completes, check your deployment status on Katabump.
