# PhotoFunia Effects Guide

## Overview
PhotoFunia provides **641+ visual effects** that you can apply to text. This document lists all available effects organized by category.

## Available Commands

### Main PhotoFunia Commands
- `.pflist` - View all available PhotoFunia effects and categories
- `.pflist <category>` - View effects in a specific category
- `.pfsearch <keyword>` - Search for effects by keyword
- `.<effect> <text>` - Apply a specific effect to your text

## Usage Examples

### Basic Text Effect
```
.neon Hello World
```
This will apply the neon effect to "Hello World"

### Frame Effect
```
.timessquare Godszeal
```
This will display "Godszeal" on a Times Square billboard

### Search for effects
```
.pfsearch neon
```
This will show all effects containing "neon" in the name

### View category
```
.pflist texteffects
```
This will show all text effects available

## Tips

1. **Text Length**: Keep your text reasonably short (1-5 words work best)
2. **Spaces in Text**: The bot handles spaces automatically
3. **Special Characters**: Most effects work with regular characters
4. **Combinations**: Try different text and effects to get unique results

---
